# Example Files

This directory contains sample security tool outputs for testing GabeApp import functionality.

## Files

- `sample_nmap.xml` - Example Nmap scan output
- `sample_nessus.nessus` - Example Nessus vulnerability scan
- `sample_zap.json` - Example OWASP ZAP web scan
- `sample_trufflehog.json` - Example TruffleHog secret detection

## Usage

1. Create a test engagement in GabeApp
2. Define scope (e.g., domains: `example.com`, IPs: `192.168.1.0/24`)
3. Import these sample files to see how the system works

## Note

These are **synthetic examples** for testing purposes only. They do not represent real vulnerabilities.
